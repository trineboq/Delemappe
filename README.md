## Obsidian
I denne mappen finner du en ".obsidian"-mappe Hvis du laster denne ned og legger den i obsidian-hvelvet ditt 
1. Sørg for at skjulte filer er synlig i Finder (Mac: Command + Shift + . (punktum))
2. Åpne obsidian-hvelv
3. Slett mappen ".obsidian"
4. Last ned mappen ".obsidian" fra denne github-siden
6. Last ned mappen "007-tmp"
8. Legg mapper inn i hvelvet
10. Åpne obsidian
11. Godkjenn på spørsmål om stoler på

Du kan nå begynne å bruke observasjonssystemet

**Tips:** Les om hvordan du bruker malen i mappen ABC1234
